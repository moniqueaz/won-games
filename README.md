# Strapi application

A quick description of your strapi application

### Genereted Plugin

```javascript
  yarn strapi generate:plugin `Name of the plugin`
```

### Genereted Service

```javascript
  yarn strapi generate:service `Name of the plugin`
```

### Data load

```shell
  curl -X POST http://localhost:1337/games/populate
```
